# Subtitle-Translator-Cli
## Description
This is a simple CLI tools to translate subtitle files.

## Usage
### Setup
add openai key to `config.json`
### Translate
```bash
stc openai -s <SOURCE_LANGUAGE> -t <TARGET_LANGUAGE> -p <PATH>
```
